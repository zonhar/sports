## 基于uniapp开发的体育彩票门店下单系统

兼容APP、小程序打包

## 支持玩法
- 竞彩足彩 N串N、 比分、半全场、进球数
- 竞彩篮球
- 足彩14场
- 任选9场
- 大乐透
- 七星彩
- 排列3  组3、和值、组6
- 排列5
- 北京单场
- 胜负过关

### 功能介绍

1. 手机号一键登录
2. 店内账本
3. 选号记录
4. 开奖
5. APP远程升级
6. 支持扫码加店
